CARCINOGÊNESE

A ORIGEM DO INIMIGO

Uma célula normal. Uma mensagem corrompida. O início de uma batalha dentro do corpo. Descobre como o cancro nasce — passo a passo, gene a gene, mutação a mutação.

// Prólogo //
O CORPO EM EQUILÍBRIO
🔬
// NARRADOR — DR. CÉLULA //
"Imagina o teu corpo como uma cidade gigante com 37 biliões de habitantes — as células. Cada uma tem uma função, um lugar, e sabe exactamente quando nascer, trabalhar e morrer. Esta ordem perfeita chama-se homeostase. Mas o que acontece quando um habitante desta cidade recusa seguir as regras?"
// conceito fundamental //
O QUE É O CANCRO?
Cancro é uma doença genética — não no sentido de "herdada", mas no sentido de que ocorre por alterações no DNA das células. Uma célula normal sofre mutações que a fazem proliferar sem controlo, invadir outros tecidos e escapar ao sistema imune.

Não é um acontecimento súbito. É um processo de décadas, com múltiplas etapas. Carcinogênese = o processo pelo qual células normais se transformam em células cancerosas.
// estatística global //
O INIMIGO EM NÚMEROS
🌍 ~20 milhões de novos casos de cancro por ano no mundo
💀 ~10 milhões de mortes anuais
🧬 2ª causa de morte global, depois das doenças cardiovasculares
⏳ O risco aumenta com a idade — a maioria dos cancros ocorre após os 50 anos
🔬 São necessárias em média 5 a 10 mutações acumuladas para que uma célula se torne maligna
// VISUALIZAÇÃO: DNA NORMAL vs DNA MUTADO //
CONCEITO-CHAVE

A carcinogênese é um processo multifásico, multigênico e clonal. Multifásico = ocorre em etapas. Multigênico = envolve vários genes. Clonal = todo o tumor deriva de uma única célula ancestral mutada.

// Capítulo 01 //
INICIAÇÃO
01
// A PRIMEIRA FAÍSCA //
INICIAÇÃO CELULAR
⚡
// NARRADOR //
"Tudo começa com um evento silencioso. Uma única célula sofre uma mutação no seu DNA — uma letra errada no código de 3 mil milhões de pares de bases. Na maioria das vezes, nada acontece. Mas quando essa mutação atinge um gene crítico... a faísca está acesa."
// definição //
O QUE É INICIAÇÃO?
A iniciação é a primeira etapa da carcinogênese. Ocorre quando um agente carcinogénicoAgente capaz de causar cancro. Pode ser químico, físico (radiação) ou biológico (vírus). causa uma alteração permanente e irreversível no DNA de uma célula — uma mutação.

Esta célula iniciada não é ainda cancerosa. É apenas... diferente. Potencialmente perigosa. Pode permanecer latente durante anos.
// características //
PROPRIEDADES DA CÉLULA INICIADA
✦ A mutação é permanente e hereditária (passa para as células-filhas)

✦ Não causa cancro por si só — precisa de promoção posterior

✦ A célula parece morfologicamente normal

✦ A iniciação requer que a célula se divida para fixar a mutação no DNA

✦ É um processo estocástico (aleatório) — ocorre por acidente

// COMO OCORREM AS MUTAÇÕES? //

☢
MUTAÇÕES DIRETAS
O agente carcinogénico ataca directamente o DNA. Ex: radiação ionizante quebra as ligações fosfodiéster. Ex: raios UV causam dímeros de pirimidina (fusão de dois T's ou C's).
⚗
ADUTOS DE DNA
Substâncias químicas (ex: benzopireno do tabaco) ligam-se covalentemente ao DNA, formando adutos. Estes bloqueiam a replicação ou causam leituras erradas.
🔁
ERROS DE REPLICAÇÃO
Durante cada divisão celular, a DNA polimerase comete ~1 erro por cada bilião de pares. Normalmente são corrigidos. Quando os sistemas de reparação falham... persistem.
🧬
INSTABILIDADE GENÓMICA
Mutações nos genes de reparação do DNA (como BRCA1, MMR) criam um estado de hipermutabilidade — a célula acumula erros muito mais depressa que o normal.
Momento 0
Exposição ao Carcinógeno
O agente atinge a célula — uma substância química, radiação ou vírus. O DNA é danificado.
Horas depois
Tentativa de Reparação
Os sistemas de reparação (NER, BER, MMR) tentam corrigir o dano. Se têm sucesso → nada acontece. Se falham → a mutação persiste.
Primeira divisão
Fixação da Mutação
A célula divide-se antes da reparação ser completa. A mutação fica fixada no DNA — permanente, hereditária. A célula está "iniciada".
Anos / décadas
Latência
A célula iniciada prolifera lentamente. Parece normal. Aguarda o próximo evento — a promoção.
ATENÇÃO CRÍTICA

A iniciação por si só não produz cancro. Para isso é necessária a fase seguinte: a promoção. Pensa assim: a iniciação acende a espoleta, mas a promoção detona a bomba. Sem promoção, muitas células iniciadas morrem naturalmente por apoptose ou senescência.

// TESTE RÁPIDO //
💡 Uma célula sofreu uma mutação num oncogene, mas o sistema de reparação falhou. A célula dividiu-se uma vez. O que acontece a seguir para que ocorra cancro?
✅ Correcto! A iniciação é apenas o primeiro passo. A mutação está fixada, mas o cancro requer múltiplos eventos moleculares adicionais — promoção e depois progressão. É um processo de décadas!
// Capítulo 02 //
PROMOÇÃO
02
// O COMBUSTÍVEL //
PROMOÇÃO TUMORAL
🔥
// NARRADOR //
"A célula iniciada estava a dormir. Mas agora chega um promotor — não um agente que muda o DNA, mas um agente que grita às células: 'Dividam-se! Crescam! Não parem!'. A espoleta está acesa. O combustível foi adicionado. A explosão aproxima-se."
// definição //
O QUE É PROMOÇÃO?
A promoção é a segunda etapa. Os agentes promotores não causam mutações no DNA — em vez disso, estimulam a proliferação das células iniciadas, amplificando a sua expansão clonal.

São geralmente reversíveis: se o promotor for removido, a proliferação pode parar. Por isso, eliminar fatores de risco (fumar, álcool, inflamação crónica) ainda pode ajudar nesta fase!
// exemplos de promotores //
AGENTES PROMOTORES
🚬 Nicotina e fenóis do tabaco → cancro do pulmão
🍺 Álcool → cancro do fígado e esófago
🌶 Ésteres de forbol (TPA) → promotor clássico experimental
🦠 Inflamação crónica (H. pylori) → cancro gástrico
🧴 Estrogénios → cancro da mama e endométrio
⚫ Sacarina (a altas doses) → promotor em bexiga de ratos
🍔 Dieta hiperlipídica → cancro colo-rectal

// MECANISMOS DA PROMOÇÃO //

📡
RECEPTORES E SINALIZAÇÃO
Promotores ligam-se a receptores celulares e activam vias como PKC (proteína-cinase C) e MAP-cinases, que levam à transcrição de genes proliferativos.
🔓
INIBIÇÃO DA APOPTOSE
Os promotores frequentemente bloqueiam a morte celular programada (apoptose), deixando as células iniciadas sobreviver mais tempo do que deveriam.
🧪
ALTERAÇÕES EPIGENÉTICAS
Muitos promotores alteram a metilação do DNA ou modificações das histonas, silenciando genes supressores de tumores sem alterar a sequência do DNA.
🔥
INFLAMAÇÃO CRÓNICA
A inflamação crónica liberta ROS (espécies reactivas de oxigénio), citocinas (TNF-α, IL-6) e factores de crescimento que promovem a expansão clonal.
DIFERENÇA CRUCIAL

🧬 Iniciação: Irreversível, modifica o DNA, ocorre uma vez, requer apenas uma exposição
🔥 Promoção: Reversível, não muda o DNA, requer exposição prolongada e repetida, pode ser interrompida

👉 Isto explica porque parar de fumar, perder peso ou tratar H. pylori reduz o risco — remove o promotor!

CÉLULA NORMAL
→
INICIADA + PROMOTOR
→
CLONE EXPANDIDO
Com promoção contínua, a clone expande exponencialmente, acumulando mais mutações

PROGRESSÃO
03
// SEM RETORNO //
PROGRESSÃO MALIGNA
💀
// NARRADOR //
"As regras foram quebradas. A célula já não está apenas a crescer mais depressa — está a acumular novas mutações a cada divisão. Como numa evolução acelerada e caótica, as mais agressivas sobrevivem. As mais invasoras dominam. Chegámos ao ponto sem retorno."
// definição //
O QUE É PROGRESSÃO?
A progressão é a terceira e mais dramática fase. O clone celular acumula mais mutações, tornando-se cada vez mais anaplástico (menos diferenciado) e maligno.

É irreversível. A célula adquire características que definem o cancro: invasão local, angiogênese, e eventualmente metástase. Os sistemas de defesa do corpo já não conseguem controlar a situação.
// evolução clonal //
SELEÇÃO DARWINIANA
Dentro do tumor, as células competem. As que têm vantagem proliferativa sobrevivem e dominam. É evolução somática — seleção natural dentro do corpo.

🏆 Sobrevivem as células com: maior velocidade de divisão, resistência à apoptose, capacidade de invadir, resistência à quimioterapia.

Crescimento local

O tumor ultrapassa a membrana basal. Para crescer além de 1-2mm, precisa de sangue: liberta VEGF para recrutar novos vasos sanguíneos (angiogênese tumoral).
Fase avançada
Metástase
A jornada mais perigosa: células separam-se (perda de E-caderina), entram na corrente sanguínea ou linfática, escapam ao sistema imune, extravasam e colonizam órgãos distantes.
// CASCATA METASTÁTICA //
7 PASSOS DA METÁSTASE
  • Invasão local da matriz extracelular (MMPs degradam colagénio)
  • Intravasação (célula entra em vaso sanguíneo ou linfático)
  • Sobrevivência na circulação (escape das NK cells)
  • Extravasação no órgão-alvo
  • Formação de micrometástase
  • Colonização e crescimento no novo local
  • Tumor secundário estabelecido
// TROPISMO METASTÁTICO //
ONDE METASTIZA CADA CANCRO?
🫁 Pulmão → fígado, osso, cérebro, supra-renais
🎗 Mama → osso, pulmão, fígado, cérebro
🟤 Cólon → fígado (via porta!), pulmão, peritóneu
🟣 Próstata → osso (predominantemente)
🖤 Melanoma → cérebro, fígado, pulmão, osso

"Teoria da semente e solo" (Paget, 1889): cada cancro tem tecidos preferidos — como sementes que só crescem em solos específicos.
// Capítulo 04 //
A GENÉTICA DO CANCRO
04
// O CÓDIGO CORROMPIDO //
ONCOGENES & SUPRESSORES
🧬
// NARRADOR //
"O cancro é fundamentalmente uma doença de genes. Não basta um único gene mutado — são precisos activar os aceleradores e destruir os travões. Dois grandes grupos de genes controlam o destino da célula: os oncogenes e os genes supressores de tumores."
// ACELERADOR //
⚡ ONCOGENES
Os proto-oncogenes são genes normais que controlam o crescimento celular. Quando sofrem mutação (ganho de função), tornam-se oncogenes — versões hiperactivas que empurram a célula a crescer sem parar.

É como um acelerador preso no carro: a célula não consegue parar.

🔑 Bastante UM alelo mutado (dominante)
// TRAVÃO //
🛑 GENES SUPRESSORES
Os genes supressores de tumores (TSGs) controlam negativamente o crescimento, activam a apoptose e reparam o DNA.

Quando ambos os alelos são inactivados (perda de função), o "travão" desaparece.

É como cortar os travões do carro — agora não há quem pare a célula.

🔑 Precisam de DOIS alelos mutados (recessivo — "Two-Hit Hypothesis")
TWO-HIT HYPOTHESIS — KNUDSON (1971)

Alfred Knudson propôs que nos cancros hereditários (ex: retinoblastoma), o primeiro "hit" é herdado (mutação germinal) e o segundo ocorre somaticamente. Nos cancros esporádicos, ambos os hits são somáticos — por isso estes são mais raros e surgem mais tarde.

📌 RB1 (retinoblastoma) foi o primeiro TSG descoberto, graças a esta hipótese.

// REDE DE SINALIZAÇÃO CELULAR INTERACTIVA //
// VIA RAS/MAPK //
A VIA MAIS MUTADA EM CANCRO
  1. Factor de crescimento liga-se ao RTK (ex: EGFR)
  2. RTK activa RAS (troca GDP por GTP)
  3. RAS activa RAF (BRAF)
  4. RAF activa MEK
  5. MEK activa ERK
  6. ERK activa factores de transcrição (MYC, FOS, JUN)
  7. Transcrição de genes de proliferação → DIVISÃO CELULAR

RAS mutado = ficou preso em "ON" → proliferação constante sem sinal externo
// VIA PI3K/AKT/mTOR //
A VIA DE SOBREVIVÊNCIA
  1. RTK activado recruta PI3K
  2. PI3K converte PIP2 → PIP3
  3. PIP3 recruta AKT (= PKB)
  4. AKT activa mTOR
  5. mTOR estimula síntese proteica e crescimento
  6. AKT também inibe apoptose (fosforila BAD, activa NF-κB)

PTEN inibe PI3K — logo PTEN perdido = AKT sempre activo = imortalidade celular
// TESTE RÁPIDO //
🧬 Um doente tem LMC (Leucemia Mieloide Crónica). A biópsia mostra a translocação t(9;22). Qual é o produto desta translocação e como causa cancro?
✅ Perfeito! O cromossoma Philadelphia (t9;22) funde BCR ao ABL, criando BCR-ABL — uma tirosina-cinase sempre activa. É por isso que o Imatinib (Gleevec) funciona: bloqueia especificamente esta cinase. Um triunfo da medicina molecular!

HALLMARK
// METABOLISMO TUMORAL //
⚡ O EFEITO WARBURG
Uma das características mais bizarras do cancro: mesmo com oxigénio disponível, as células cancerosas preferem fazer glicólise anaeróbica em vez de respiração oxidativa.

🍬 Normal: 1 glucose → 36 ATP (respiração)
🔥 Cancro: 1 glucose → 2 ATP (glicólise) — mas muito mais rápido!

Porquê? A glicólise fornece precursores biossintéticos para construir biomassa (membranas, nucleotídeos, aminoácidos) — essencial para crescimento rápido.

Base do PET scan: tumores captam mais 18F-FDG (análogo de glucose) que tecidos normais.
// IMUNOEVASÃO //
🛡 COMO O CANCRO ENGANA O SISTEMA IMUNE
O sistema imune devia destruir células anormais. O cancro evita isso através de:

◉ Downregulation de MHC-I → células NK não reconhecem o tumor
◉ Expressão de PD-L1 → bloqueia células T citotóxicas
◉ TGF-β e IL-10 → suprimem resposta imune local
◉ Recrutamento de Tregs e MDSCs → criam microambiente imunossupressor
◉ Perda de antígenos tumorais → o sistema imune não vê nada para atacar

Base da imunoterapia: bloqueadores de PD-1/PD-L1 (pembrolizumab, nivolumab) reactivam a resposta imune.

OS AGENTES DO CRIME
06
// CARCINÓGENOS //
QUEM INICIA A GUERRA?
☠
// NARRADOR //
"Nem toda a mutação é causada por acidente. Existem agentes — químicos, físicos e biológicos — que empurram activamente o DNA para o caos. São os carcinógenos. Conhecê-los é o primeiro passo para a prevenção."
⚗
CARCINÓGENOS QUÍMICOS
A maioria requer activação metabólica pelo CYP450 hepático (pró-carcinógenos → carcinógenos activos).

▸ Hidrocarbonetos aromáticos policíclicos (HAPs) — tabaco, carne grelhada
▸ Aminas aromáticas — corantes industriais → cancro da bexiga
▸ Aflatoxina B1 (fungo Aspergillus) → mutação TP53 codon 249 → CHC
▸ Nitrosaminas — carnes processadas, fumo
▸ Benzeno → leucemia
▸ Asbesto → mesotelioma, pulmão (efeito sinérgico com tabaco!)
☢
CARCINÓGENOS FÍSICOS
Radiação ionizante (raios X, γ, partículas α/β/nêutrons): Causa quebras de dupla hélice, translocações cromossómicas, deleções. Ex: Hiroshima → leucemia. Chernobyl → cancro da tireoide.

Radiação UV (UV-B, 280-315nm): Cria dímeros de ciclobutano-pirimidina (CPD) → mutações características C→T em TP53 → melanoma, carcinoma basocelular.

Radão (gás natural radioactivo) → 2ª causa de cancro de pulmão em não-fumadores!
🦠
VÍRUS ONCOGÉNICOS
HPV (16 e 18): E6 degrada p53; E7 bloqueia Rb → cancro do colo do útero, orofaringe

HBV/HCV: inflamação crónica + cirrose → Carcinoma hepatocelular

EBV: EBNA-2 activa MYC → Linfoma de Burkitt, Hodgkin, NPC

HTLV-1: proteína Tax activa NF-κB → leucemia/linfoma de células T

HHV-8 (KSHV): codifica homólogos de ciclina D → Sarcoma de Kaposi
🦠
BACTÉRIAS E PARASITAS
H. pylori: Causa inflamação crónica gástrica → activa NF-κB → citocinas → cancro gástrico. CagA injecta proteína oncogénica via sistema de secreção tipo IV!

Schistosoma haematobium: Inflamação crónica na bexiga → carcinoma de células escamosas da bexiga

Opisthorchis viverrini: Nos ductos biliares → colangiocarcinoma
CASO CLÍNICO MOLECULAR: AFLATOXINA B1

🌽 A aflatoxina B1 (produzida pelo fungo Aspergillus flavus em alimentos mal armazenados) é um dos carcinógenos mais potentes conhecidos.

⚙️ Mecanismo: CYP3A4 no fígado converte AFB1 → AFB1-8,9-epóxido → liga-se covalentemente à guanina do DNA (aduto) → durante replicação, causa substituição G→T especificamente no codão 249 do TP53.

📍 Esta mutação "assinatura" (R249S) é encontrada em 50-60% dos CHC em regiões endémicas de África subsaariana e Ásia, onde o consumo de alimentos contaminados com aflatoxina é comum. Esta é uma das provas mais elegantes da etiologia molecular do cancro.

// CICLO CELULAR E CHECKPOINTS //
// CHECKPOINTS DO CICLO CELULAR //
OS PORTÕES DE CONTROLO
G1/S checkpoint: Verifica tamanho, dano no DNA, nutrientes. Rb bloqueia E2F. Se DNA danificado → p53 activa → p21 inibe CDK2 → paragem.

Intra-S checkpoint: Monitoriza replicação. ATR e ATM detectam forquilhas paradas ou DSBs → CHK1/CHK2 → inibem CDKs.

G2/M checkpoint: Garante que o DNA foi completamente e correctamente replicado antes da mitose.

Spindle checkpoint (SAC): Garante que cada cromossoma está correctamente ligado ao fuso mitótico. Previne aneuploidia.
// CDKs E CICLINAS //
OS ACELERADORES DO CICLO
As ciclinas activam as CDKs (cinases dependentes de ciclina) que impulsionam o ciclo:

▸ Ciclina D + CDK4/6 → fosforila Rb → liberta E2F → genes S-fase
▸ Ciclina E + CDK2 → transição G1→S
▸ Ciclina A + CDK2 → replicação do DNA (S-fase)
▸ Ciclina B + CDK1 (CDC2) → entrada em mitose

Em cancro: ciclina D1 sobreexpressa (amplificação CCND1), CDK4 amplificado, p16 (CDKN2A) inactivado → Rb é fosforilado e inactivo → divisão descontrolada.

// APOPTOSE — O SUICÍDIO PROGRAMADO //

// VIA INTRÍNSECA //
MITOCONDRIAL
Activada por: dano no DNA, hipóxia, factores de crescimento reduzidos, stress do RE

Família BCL-2 controla esta via:
▸ Pro-apoptóticos: BAX, BAK, BID, BIM, PUMA, NOXA
▸ Anti-apoptóticos: BCL-2, BCL-XL, MCL-1

Quando pro-apoptóticos dominam → permeabilização da membrana mitocondrial externa → libertação de citocromo c → formação do apoptossoma (citocromo c + APAF-1 + Caspase-9) → activação de Caspase-3 → MORTE CELULAR

BCL-2 sobreexpresso em Linfoma folicular [t(14;18)]: a célula cancerosa é imortal porque BCL-2 bloqueia BAX/BAK.
// VIA EXTRÍNSECA //
RECEPTORES DE MORTE
Activada por: FasL (células T citotóxicas), TNF-α, TRAIL

FasL liga-se a Fas (CD95) → trimerização → recrutamento de FADD → recruta Caspase-8 → forma o DISC (complexo indutor de morte) → Caspase-8 activa-se → activa Caspase-3 → MORTE CELULAR

▸ Caspase-8 também pode clivar BID → tBID → activa via mitocondrial (cross-talk)

Como o cancro escapa:
◉ Downregulation de Fas/TRAIL-R
◉ Sobreexpressão de FLIP (inibe Caspase-8)
◉ Sobreexpressão de XIAP (inibe caspases)
◉ Mutação/perda de p53 → PUMA e NOXA não são transcritos
DESTAQUE: TP53 — O GUARDIÃO DO GENOMA

p53 é a proteína mais importante na prevenção do cancro. É mutada em ~50% de TODOS os cancros humanos.

Em condições normais: p53 tem semi-vida curta (~20 min) — é continuamente ubiquitinada pela MDM2 e degradada pelo proteasoma.

Quando o DNA é danificado: ATM/ATR fosforilam p53 → impede ligação a MDM2 → p53 estabiliza → activa transcrição de:
▸ p21 (CDKN1A): inibe CDK2 → para em G1
▸ GADD45: reparação do DNA
▸ BAX, PUMA, NOXA: apoptose
▸ p16: senescência

Se o dano é irreparável → p53 força apoptose. Se é reparável → p53 para o ciclo para dar tempo.

Síndrome de Li-Fraumeni: Mutação germinal de TP53 → múltiplos cancros precoces (sarcomas, mama, cérebro, leucemias) — demonstra que TP53 é verdadeiramente o guardião do genoma.

// SENESCÊNCIA E TELÓMEROS //
A BARREIRA MORTAL
Cada divisão celular encurta os telómeros (~50-200 pb por divisão). Após ~50-70 divisões (Limite de Hayflick), os telómeros ficam criticamente curtos → sinais de dano no DNA → senescência (paragem permanente) ou apoptose.

Para se tornar imortal, o cancro precisa de manter os telómeros:
▸ Telomerase (hTERT): reactivada em 85-90% dos cancros — adiciona TTAGGG repetidamente
▸ ALT (Alternative Lengthening of Telomeres): recombinação homóloga — nos restantes 10-15%

A telomerase é alvo terapêutico promissor — é activa no cancro mas quase ausente em células somáticas normais.
// ANGIOGÊNESE TUMORAL //
RECRUTAR VASOS: O ANGIOGENIC SWITCH
Um tumor > 1-2mm precisa de vascularização própria. Activa o "angiogenic switch":

Pró-angiogénicos ↑: VEGF-A (principal), bFGF, PDGF, angiopoietinas
Anti-angiogénicos ↓: Trombospondina-1, angiostatina, endostatina

Mecanismo: Hipóxia tumoral → HIF-1α estabiliza (sem degradação por VHL) → transcreve VEGF → VEGF liga VEGFR2 em células endoteliais → proliferação, migração → novos vasos caóticos, imaturos, permeáveis

Bevacizumab (anti-VEGF) e sorafenib (anti-VEGFR) são antiangiogénicos usados clinicamente.
// TESTE FINAL //
🔬 Um doente com Síndrome de Lynch tem mutação germinal no gene MSH2. Qual o mecanismo pelo qual isso leva a cancro colo-rectal?
✅ Excelente! O MMR (sistema de reparação por excisão de mismatch) corrige erros introduzidos pela DNA polimerase. Sem MSH2, os microssatélites tornam-se instáveis (MSI-H) e mutações acumulam-se rapidamente em genes críticos. O diagnóstico é feito por MSI testing e imunohistoquímica. A imunoterapia (anti-PD-1) é especialmente eficaz em tumores MSI-H!

CRONOLOGIA DA CARCINOGÊNESE
📚
// NARRADOR — EPILOGO //
"Agora tens o mapa completo. Uma célula normal → uma faísca de mutação → anos de promoção → acumulação de erros → ruptura das defesas → invasão → metástase. Décadas de batalha silenciosa. Mas compreender é o primeiro passo para vencer."
Ano 0 — Célula Normal
Homeostase Perfeita
Ciclo celular regulado, apoptose funcional, telómeros intactos, DNA íntegro. Todos os checkpoints operacionais.
Ano 0 → Anos 5-10 — INICIAÇÃO
1ª Mutação Crítica
Exposição a carcinógeno (tabaco, UV, aflatoxina, vírus...). Mutação num proto-oncogene (ex: RAS) ou TSG (ex: APC). Célula "iniciada" — silenciosa mas alterada.
Anos 5-20 — PROMOÇÃO
Expansão Clonal
Exposição prolongada a promotores. Célula iniciada prolifera → clone de milhões de células. Ainda reversível se o promotor for removido. Displasia pode ser visível histologicamente.
Anos 15-30 — PROGRESSÃO
Acumulação de Mutações
Instabilidade genómica. Novas mutações em TP53, PTEN, BRCA... Células mais agressivas selecionadas. Aparecimento de anaplasia, carcinoma in situ.
Anos 25-40 — MALIGNIDADE
Cancro Invasivo
Ruptura da membrana basal. Angiogênese tumoral (VEGF). Evasão ao sistema imune (PD-L1). O tumor é agora clinicamente detectável. Sintomas podem surgir.
Fase final — METÁSTASE
Disseminação Sistémica
Células invadem vasos, circulam, extravasam. Colonização de órgãos distantes. Prognóstico grave. Base para terapias sistémicas (quimio, imuno, targeted therapy).
// RESUMO DAS ETAPAS //
AS 3 FASES EM SEGUNDOS
INICIAÇÃO Mutação permanente no DNA. Irreversível. Uma exposição. Célula parece normal.
PROMOÇÃO Proliferação do clone. Reversível. Exposição prolongada necessária. Displasia.
PROGRESSÃO Mais mutações, invasão, metástase. Irreversível. Evolução clonal darwiniana.
